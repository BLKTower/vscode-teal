# Teal for Visual Studio Code

Provides a language server and syntax highlighting for [Teal](https://github.com/teal-language/tl) in VSCode.

## Features

- Syntax highlighting
- Error checking

## Requirements

Make sure that the Teal compiler is available in your PATH:
```
luarocks install tl
```
