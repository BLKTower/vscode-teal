# TL support for Visual Studio Code

Provides a language server and syntax highlighting for [TL](https://github.com/hishamhm/tl) in VSCode.

## Features

- Syntax highlighting
- Error checking

## Requirements

Make sure that TL is available in your path:
```
luarocks install tl
```
